About Interlacer

Interlacer combines multiple images and colors into a single image by interlacing them together. The final image consists of a line from each image/color in succession, like this:

Using Interlacer

Tap the button to add images or colors. You can add images from the camera or from your photo albums. As you add images and colors, the interlaced image will be processed and displayed in the preview area.

The size of the interlaced image will be the smallest width/height of all the images that you have selected. If you select only color swatches, the size of the interlaced image will be the screen resolution of your device.

The preview image is a scaled down version of the full size image. To see the image at full resolution, save it and look at it in the Photos App.

Tap to save the image. If you want to start over, tap to clear your selections. This will clear all the image and color selections and the preview area, but will not reset the interlace setting sliders.

Tap on any image or color swatch to reveal the image detail view. If the selection is an image, you will see the size of the image. If the selection is a color, you will see the RGB values for the color. You can also remove the image/color by tapping the button on the image detail view.

Interlace Settings

Tap the button to adjust the interlace settings. By default, one line from each image/color is used at a time. You can make the processed image look chunkier by adjusting the line size. Sliding the line size all the way to the right will enable random mode, where the line size will vary randomly.

Adjusting the shift setting will cause every other line in the image to be shifted a certain number of pixels. Try selecting only one image and adjust the shift to get an idea of what it does.

Tap to reset the sliders to their default values.

App Settings

You can change application settings by using the Settings App, or by tapping the icon in the upper right corner of the screen. Available application settings are:

  • Photo Overlay: If Photo Overlay is turned on, the first selected image will be overlaid on the camera view. This can assist with lining up multiple images.
  • Save Camera Photos: If Save Camera Photos is turned on, images you take with the camera will automatically be saved.
  • Autosave Results: If Autosave Results is turned on, each interlaced image will be saved as it is created. Be careful with this one. You may end up saving more images than you realize!

FAQ

Why did it crash? It crashed because you found a bug. :-) Efforts have been made to protect against many conditions that could cause a crash. However, it is nearly impossible to test every scenario. If you do experience a crash, please sync your device with iTunes. The crash report will be submitted to the App Store, where the developer can view it and use it to improve the application.

Why doesn't it do x, y, and z? Send the developer an email with your feature request. If enough people ask for a feature, chances are it will get implemented in a future update!

How many images can I use at once? The number of images you can process is limited by the amount of memory that is free for the application to use and the size of the images you are trying to interlace. During testing, dozens of images were successfully interlaced.

Is there a Flickr group for the app? Yes, there is! Feel free to submit any images you create with Interlacer to the Interlacer App Flickr group, located at http://www.flickr.com/groups/interlacerapp/.
